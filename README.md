# PinSign – Consultation signalétique participative

## Déploiement pas à pas

### 1. Déposer le projet sur GitHub

1. Allez sur [github.com](https://github.com) et connectez-vous
2. Cliquez sur le **"+"** en haut à droite → **"New repository"**
3. Nom : `pinsign` · Laissez tout le reste par défaut · Cliquez **"Create repository"**
4. Sur la page qui s'affiche, cliquez **"uploading an existing file"**
5. Glissez-déposez **tous les fichiers de ce dossier** (en respectant la structure)
6. Cliquez **"Commit changes"**

### 2. Déployer sur Vercel

1. Allez sur [vercel.com](https://vercel.com) → **"Sign up"** avec votre compte GitHub
2. Cliquez **"Add New Project"**
3. Sélectionnez le repository `pinsign`
4. Vercel détecte automatiquement que c'est un projet React
5. Cliquez **"Deploy"** → attendez ~2 minutes
6. Votre site est en ligne ! Vercel vous donne une URL du type `pinsign.vercel.app`

### 3. Utilisation

- **Charger le plan** : cliquez "Charger un plan" et importez votre image JPG/PNG
- **Partager** : copiez l'URL et envoyez-la à vos participants
- **Résultats** : cliquez "Résultats" pour voir toutes les propositions et les témoignages
